### 踩的坑
1. **Astro 7 中 `post.render()` 已改为 `render(post)`**（从 `astro:content` 导入）
2. **Astro 组件模板中使用的组件必须显式 import**（不能省略）
3. **`rss.xml.ts` 不能用 `---` frontmatter 分隔符**（这是 `.astro` 文件的语法，`.ts` 文件不行）
4. **TypeScript 6.x 与 Astro 7 有兼容问题** → 降级到 TypeScript 5.8.3
5. **Astro telemetry 在沙箱会导致 `EPERM` 错误** → 用 `ASTRO_TELEMETRY_DISABLED=1` 禁用
